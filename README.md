# simpcreate 简幻欢实例创建&变配工具

## 📋 项目概述
简幻欢实例创建&变配工具是一个用于快捷创建或变配简幻欢实例的网页工具。该工具提供了直观的界面，帮助用户轻松管理和操作简幻欢实例，包括获取实例信息、创建新实例、实例变配等功能。

## ✨ 功能特点

### 🔒 Token 管理
- 支持从简幻欢官网获取 Token
- 将 Token 安全保存到本地存储，方便后续使用
- 提供 Token 清除功能，保护用户隐私

### 📊 配置与实例管理
- **配置列表加载**：从简幻欢拉取可用的实例配置列表，以表格形式直观展示
- **用户实例列表**：拉取用户的实例列表，支持查看详细配置信息
- **配置排序**：支持单排序和双重排序模式，可按不同列进行升降排序
- **实例状态显示**：清晰展示实例空余情况

### 🚀 实例创建
- **单次创建**：选择空余配置，一键创建实例
- **循环创建**：选择不空余实例，自动重复尝试创建直到成功
- **创建间隔设置**：可自定义循环创建的时间间隔
- **创建进度显示**：实时显示循环创建的进度和尝试次数

### 🔄 实例变配
- **单次变配**：选择自己的实例和空余配置，一键变配
- **自动变配**：选择不空余实例，自动重复尝试变配直到成功
- **开服功能**：支持直接在工具中开启实例

### 🎨 界面与体验
- 响应式设计，适配不同屏幕尺寸
- 现代化的 UI 设计，直观易用
- 实时通知系统，操作结果即时反馈
- 加载状态提示，提升用户体验
- 支持 CPU 等级说明文档快速访问

## 📖 使用说明

### 🔑 首次使用

1. **访问网站**：打开 [简幻欢实例创建&变配工具](https://simpcreate.ip-ddns.com/)

2. **获取 Token**：
   - 将"简幻欢Token获取"按钮拖拽到浏览器收藏夹栏
   - 访问 [简幻欢官网](https://simpfun.cn/console) 并登录您的账号
   - 点击刚才添加的收藏夹按钮，将 Token 复制到剪贴板

3. **提交 Token**：
   - 在工具页面的输入框中粘贴您的 Token
   - 点击"提交 Token"按钮

4. **加载配置列表**：提交 Token 后，工具会自动加载可用的实例配置列表

### 🔄 后续使用
如果您已经保存了 Token，打开页面时会自动加载配置列表。

### 🎯 创建实例

1. **选择配置**：在配置列表中选择一个合适的配置（绿色"可创建"状态）
2. **选择创建模式**：
   - **单次创建**：点击"立即创建实例"按钮，发送一次创建请求
   - **循环创建**：
     - 选择"循环创建"模式
     - 设置请求间隔时间（建议不小于30秒）
     - 点击"开始循环创建"按钮，系统会自动重复尝试创建直到成功

### 🔄 实例变配

1. **切换到实例模式**：点击"我的实例模式"按钮
2. **选择实例**：在实例列表中选择您要变配的实例
3. **选择目标配置**：在配置列表中选择一个目标配置
4. **执行变配**：
   - 如果目标配置可创建，点击"执行变配"按钮
   - 如果目标配置不可创建，点击"开始自动变配"按钮，系统会自动重复尝试变配直到成功

5. **开服操作**：变配完成后，可点击"开服"按钮启动实例
