# simpcreate 简幻欢(simpfun)实例创建&变配工具

## 📋 项目概述
简幻欢实例创建&变配工具是一个专业的第三方辅助工具，旨在为简幻欢 (Simpfun) 用户提供比官网更高效、更直观的实例管理体验。它支持自动抢占实例、多规格轮询尝试、双重排序过滤以及实时的 CPU 型号映射，是您管理简幻欢实例的强力助手。

[访问工具地址](https://simpcreate.ip-ddns.com/)

## ✨ 核功能与亮点

### 🚀 实例抢占与自动化
- **循环创建/变配**：针对不空余的配置，可设置自定义间隔（最低 10s）自动重复尝试，抢占心仪规格。
- **多配置队列尝试 (NEW)**：支持构建“多配置队列”，程序将按优先级由高到低依次尝试队列中的规格，极大提升抢占成功率。
- **循环开机**：针对因资源紧张无法启动的实例，支持循环发送开机请求，直到成功开启。
- **进度可视化**：所有自动化任务均配备进度条和尝试计数器，直观掌控任务状态。

### 📊 高级表格管理
- **自由缩放表格**：配置列表与队列列表均支持右下角点击拖拽缩放，完美适配不同屏幕。
- **双重排序系统**：支持“单排序”与“双重排序”切换。在双重模式下，可同时选定主、次两个排序列（如：先按 CPU 等级排，再按价格排）。
- **CPU 规格直观显示**：实时将简幻欢的字母等级（C+, B- 等）映射为具体的 CPU 型号（如 E5-2682v4, 9950X, U7-265K），并显示 CPU-Z 参考跑分。
- **一键复制**：点击用户名、UID 等关键信息即可快速复制。

### 🔒 安全与便捷
- **Token 管理**：支持通过书签脚本（Bookmarklet）一键获取官网 Token，并安全存储于本地。
- **多模式切换**：灵活切换“创建实例模式”与“我的实例模式”，互不干扰。
- **响应式 UI**：基于现代化的 UI 设计，适配手机与 PC 浏览器，支持流畅的过渡动画。

## 📖 使用说明

### 🔑 快速开始 (Token 设置)
1. **获取脚本**：将页面顶部“简幻欢Token获取”按钮拖动到浏览器收藏夹栏。
2. **提取 Token**：登录 [简幻欢官网](https://simpfun.cn/console)，点击刚才添加的收藏夹书签。
3. **提交验证**：将复制的 Token 粘贴到本工具的输入框并提交。
4. **持久化**：Token 将自动保存，下次访问无需重复输入（除非手动清除）。

### 🎯 两种主要模式
- **创建实例模式**：
    - 点击列表中的配置即可选中。
    - **单次创建**：直接发送请求。
    - **循环创建**：针对“不可创建”配置，设定间隔自动刷。
    - **多配置**：点击表格中的配置行将其加入下方队列，程序将循环重试队列中的所有项（优先级从上往下）。
- **我的实例模式**：
    - 查看自己所有实例的详细信息（流量详情、端口、状态）。
    - 支持**循环开机**，解决抢占后无法启动的尴尬。
    - 支持**实例变配**：选中实例后，在上方列表中点击目标配置，同样支持循环和多配置逻辑。
